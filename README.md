## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- nodejs 12.x
- npm - to install packages locally

### Setup

- git clone https://github.com/rbanos-mv/capstone-1-expo.git

### Install

- cd capstone-1-event
- npm install

### Usage

- Lint code
  - npm run webhint
  - npm run stylelint
